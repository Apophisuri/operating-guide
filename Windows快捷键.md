# Windows快捷键

## 一、系统级操作（最核心）

| 功能               | 快捷键               |
| ------------------ | -------------------- |
| 打开设置           | `Win + I`            |
| 打开文件资源管理器 | `Win + E`            |
| 打开运行           | `Win + R`            |
| 打开任务管理器     | `Ctrl + Shift + Esc` |
| 锁屏               | `Win + L`            |
| **显示桌面**       | **`Win + D`**        |
| 切换窗口           | `Alt + Tab`          |
| 关闭当前窗口       | `Alt + F4`           |
| 截图工具           | `Win + Shift + S`    |

------

## 二、窗口管理（提升效率关键）

| 功能             | 快捷键                |
| ---------------- | --------------------- |
| 最大化窗口       | `Win + ↑`             |
| 最小化窗口       | `Win + ↓`             |
| 窗口贴左         | `Win + ←`             |
| 窗口贴右         | `Win + →`             |
| 多屏移动窗口     | `Win + Shift + ← / →` |
| **切换桌面**     | `Ctrl + Win + ← / →`  |
| **新建虚拟桌面** | `Ctrl + Win + D`      |
| **关闭虚拟桌面** | `Ctrl + Win + F4`     |

------

## 三、文本编辑（必须肌肉记忆）

| 功能 | 快捷键     |
| ---- | ---------- |
| 复制 | `Ctrl + C` |
| 粘贴 | `Ctrl + V` |
| 剪切 | `Ctrl + X` |
| 撤销 | `Ctrl + Z` |
| 重做 | `Ctrl + Y` |
| 全选 | `Ctrl + A` |
| 查找 | `Ctrl + F` |
| 保存 | `Ctrl + S` |

 进阶（非常实用）：

- 按词移动：`Ctrl + ← / →`
- 删除一个词：`Ctrl + Backspace`

------

## 四、文件操作（高频）

| 功能           | 快捷键             |
| -------------- | ------------------ |
| **新建文件夹** | `Ctrl + Shift + N` |
| 重命名         | `F2`               |
| 删除           | `Delete`           |
| 彻底删除       | `Shift + Delete`   |
| **属性**       | `Alt + Enter`      |
| 刷新           | `F5`               |

------

## 五、浏览器通用快捷键

（适用于 Chrome / Edge）

| 功能         | 快捷键             |
| ------------ | ------------------ |
| 新标签页     | `Ctrl + T`         |
| 关闭标签页   | `Ctrl + W`         |
| 恢复关闭标签 | `Ctrl + Shift + T` |
| 切换标签     | `Ctrl + Tab`       |
| 地址栏       | `Ctrl + L`         |
| 强制刷新     | `Ctrl + Shift + R` |

------

## 六、输入效率（隐藏高频）

| 功能       | 快捷键                    |
| ---------- | ------------------------- |
| 切换输入法 | `Win + Space`             |
| 切换中英文 | `Shift` 或 `Ctrl + Space` |
| 打开 Emoji | `Win + .`                 |

------

## 七、开发/高级用户

| 功能                         | 快捷键                            |
| ---------------------------- | --------------------------------- |
| 打开终端（Windows Terminal） | `Win + X → 选择`                  |
| 直接终端                     | `Win + R → 输入 cmd / powershell` |
| 管理员菜单                   | `Win + X`                         |
| 锁定任务栏应用启动           | `Win + 数字键`                    |
