# macOS 使用指南

在 macOS：
- Command（⌘）是主操作键
- Option（⌥）是功能增强键
- Control（⌃）使用较少

---

## 窗口管理 → 应用管理

Windows：
- 操作的是“窗口”

macOS：
- 操作的是“应用（App）”

区别：

| 行为     | Windows   | macOS             |
| -------- | --------- | ----------------- |
| 关闭窗口 | 退出程序  | 仅关闭窗口        |
| 切换     | Alt + Tab | ⌘ + Tab（应用级） |

关键快捷键：
- ⌘ + Q：退出应用
- ⌘ + W：关闭窗口

---

## 3. 没有“剪切文件”

macOS 不支持 Finder 中的剪切操作

移动文件方式：

```cmd
⌘ + C → ⌘ + Option + V
```

---

## 找不到文件路径

解决：
Finder 中：

- ⌘ + Option + P：显示路径栏
- 右键窗口标题：查看路径

# 核心快捷键

## 1. 基础操作

| 功能         | 快捷键 |
| ------------ | ------ |
| 复制         | ⌘ + C  |
| 粘贴         | ⌘ + V  |
| 剪切（文本） | ⌘ + X  |
| 撤销         | ⌘ + Z  |
| 保存         | ⌘ + S  |
| 全选         | ⌘ + A  |

---

## 2. 应用与窗口

| 功能     | 快捷键           |
| -------- | ---------------- |
| 切换应用 | ⌘ + Tab          |
| 切换窗口 | ⌘ + `            |
| 关闭窗口 | ⌘ + W            |
| 退出应用 | ⌘ + Q            |
| 强制退出 | ⌘ + Option + Esc |

---

## 3. Finder 文件操作

| 功能       | 快捷键                 |
| ---------- | ---------------------- |
| 新建窗口   | ⌘ + N                  |
| 新建文件夹 | ⌘ + Shift + N          |
| 删除       | ⌘ + Delete             |
| 移动文件   | ⌘ + C → ⌘ + Option + V |

---

## 4. 截图

| 功能     | 快捷键        |
| -------- | ------------- |
| 全屏截图 | ⌘ + Shift + 3 |
| 区域截图 | ⌘ + Shift + 4 |
| 截图工具 | ⌘ + Shift + 5 |

---

## 5. 搜索

- ⌘ + Space：Spotlight

用途：
- 打开应用
- 查找文件
- 计算
- 搜索信息



## 选中文本

```txt
Shift + ⌘ + ←   （选中从光标到行首）
Shift + ⌘ + →   （选中从光标到行尾）

⌘ + L

Shift + ↑ / ↓   （按行扩展）
Shift + ⌘ + ↑   （选到文档开头）
Shift + ⌘ + ↓   （选到文档结尾）


```
